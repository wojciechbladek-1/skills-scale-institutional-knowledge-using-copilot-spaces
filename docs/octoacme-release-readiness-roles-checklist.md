# OctoAcme — Release Readiness Roles Checklist

Use before production deployment to align quality and release coordination responsibilities.

## QA Lead readiness checks
- [ ] Test execution complete for release scope (including critical regressions)
- [ ] Open defects triaged with severity and release impact
- [ ] Quality recommendation documented (go / conditional go / no-go)
- [ ] Post-release verification checks prepared

## Release Manager readiness checks
- [ ] Release window and change communication confirmed
- [ ] Deployment runbook and rollback plan validated
- [ ] Go/no-go review held with required stakeholders
- [ ] Stakeholder and support announcement draft prepared

## Shared release controls
- [ ] Final scope and release notes confirmed with Product Manager
- [ ] Technical readiness confirmed with Tech Lead / Engineering Manager
- [ ] Incident escalation path validated with Project Manager and on-call team
- [ ] Post-release ownership for monitoring and follow-up actions assigned
